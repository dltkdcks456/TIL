## git **help**

Git 사용 중 모르는 부분이 있을 때 도움을 받을 수 있는 기능

```
git help
```

- 기본적인 명령어들과 설명



```
git help -a
```

- Git의 모든 명령어들
- `j`로 내리기, `k`로 올리기, `:q`로 닫기



```
git (명령어) -h
```

- 해당 명령어의 설명과 옵션 보기



```
git help (명령어)
```

 

```
git (명령어) --help
```

- 해당 명령어의 설명과 옵션 웹사이트에서 보기
- ⭐️ 웹에서 열리지 않을 시 끝에 `-w`를 붙여 명시
