# 컴퓨터의 자료표현

- 비트(Bit)
  - 컴퓨터가 값을 저장할 수 있는 최소 단위(0, 1)

- 바이트(Byte)
  - 1바이트 = 8bit
  - 정보를 처리하기 위한 최소 단위는 바이트를 사용

- 용량단위
  - SI 접두어
    - 10^3을 기준으로 단위를 붙임(K, M, G, T)
  - IEC
    - 2^10(= 1024)을 기준(Ki, Mi, Gi, Ti)

> 자바에서는 1바이트를 0 ~ 255를 사용하는 게 아닌 -128 ~ 127까지의 수를 사용
>
> 컴퓨터는 음수를 나타낼 때 2의 보수법을 사용
